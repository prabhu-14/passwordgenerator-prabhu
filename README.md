# passwordgenerator-prabhu

- A simple and secure Command Line Password generator built with python
- This project generates strong random passwords.

🎯 Objective
- Genereate secure and random passwords directly from the command line

✨ Features
✅Random password generation
✅Includes uppercase letters(A-Z)
✅Includes lowercase letters(a-z)
✅Includes numbers(0-9)
✅Includes special characters(@#$%*)
✅User defined password length
✅Strong and secure passwords

🛠️ Technologies used
• Python
• random module
• string module

📍 Requirements
- Make sure python is installed.
